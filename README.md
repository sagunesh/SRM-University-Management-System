# SRM University Management System
It is an online college intranet system which has profile sharing,private messaging system,notes sharing facilities and also contains get notified facility  as your teacher uploads an assignment.You can also get all updates of your college like daily news, other  activities and daily events that are happening or going to happen in your college.
